# dart_lmdb2

This project contains a Dart package to use LMDB from Dart via `dart::ffi`.

The main package in this repository is [dart_lmdb2](./dart_lmdb2/README.md), which contains the LMDB source code, the Dart API wrappers and can be used in Dart-only projects.

The flutter package [flutter_lmdb2](./flutter_lmdb2/README.md) is a Flutter wrapper around `dart_lmdb2` to be able to use it on most Flutter platforms.
